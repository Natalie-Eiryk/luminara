## Part 1: Visual Identity & Polish

### 1.1 Dynamic Backgrounds & Parallax

**Concept**: Each category has a living, breathing environment that reacts to gameplay.

```
CATEGORY BIOMES:

000 - Cellular:
  - Floating organelles, mitochondria pulsing with energy
  - Phospholipid membrane waves in background
  - Ribosomes constructing proteins (particle effects)
  - Color palette: Bioluminescent greens/cyans

100 - Neural:
  - Synapse lightning storms
  - Neurotransmitter rain (dopamine = gold, serotonin = purple)
  - Axon highways with traveling signals
  - Color palette: Electric purples/blues

200 - Sensory:
  - Eye motifs that track cursor
  - Sound wave visualizations
  - Tactile texture morphing
  - Color palette: Rainbow spectrum shifts

400 - Tissue:
  - Muscle fiber contractions
  - Collagen weaving animations
  - Healing/regeneration particles
  - Color palette: Warm reds/pinks

500 - Organ Systems:
  - Heartbeat camera pulse
  - Breathing background expansion
  - Blood flow particle streams
  - Color palette: Deep crimsons

REACTIVE ELEMENTS:
- Correct answer: Environment brightens, particles celebrate
- Wrong answer: Environment dims, distortion effects
- Combo streak: Background intensifies, speed increases
- Boss fight: Storm effects, dramatic lighting
```

## Part 2: Advanced Game Modes

### 2.1 "Anatomy Theater" (Rhythm Game + Quiz)

**Concept**: Answer questions to the beat. Timing affects damage/rewards.

```
RHYTHM MECHANICS:

BEAT ZONES:
  ├── PERFECT ──┤ (±50ms) = 2x damage, 2x XP
  ├── GREAT ────┤ (±100ms) = 1.5x damage
  ├── GOOD ─────┤ (±200ms) = 1x damage
  └── MISS ─────┘ = 0.5x damage, combo break

VISUAL RHYTHM:
- Question appears with pulse indicator
- Options have approach timing circles
- Background pulses to BPM
- Monster attacks sync to off-beats

DIFFICULTY MODES:
- Adagio (60 BPM): Relaxed, forgiving timing
- Allegro (90 BPM): Standard challenge
- Presto (120 BPM): Expert speed
- Fortissimo (150 BPM): Insane mode

BOSS BATTLES:
- Boss has "attack patterns" synced to music
- Player must answer during "openings"
- Phase transitions = tempo changes
- Final phase = dramatic orchestral swell
```

### 2.3 "Surgical Simulator" (Precision Mini-Games)

**Concept**: Apply knowledge in simulated procedures. Questions unlock tools/steps.

```
PROCEDURE FLOW:

1. CASE PRESENTATION
   - Patient history, symptoms
   - Must identify likely diagnosis (quiz)

2. PREPARATION
   - Select correct instruments (matching)
   - Identify anatomy landmarks (labeling)

3. PROCEDURE
   - Step-by-step guidance
   - Wrong answers = complications
   - Time pressure element

4. CLOSURE
   - Post-op questions
   - Outcome based on performance

MINI-GAME TYPES:
- Incision Planning: Trace correct path
- Vessel Identification: Color-code arteries/veins
- Nerve Avoidance: Navigate without touching
- Suturing: Rhythm-based closing
- Diagnosis: Symptom → condition matching

GRADES:
- Perfect Surgery: No complications, fast time
- Successful: Minor issues, completed
- Complicated: Multiple problems
- Critical: Major errors (learning opportunity)
```

NEURAL TOWERS:

TISSUE TOWERS:

ENEMY WAVES:

RESOURCE GENERATION:
